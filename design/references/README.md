# Design References

This folder holds reference screenshots and mockups used as visual direction for the Expertly portal.

---

## How to use

1. **Drop screenshots here** — name them clearly (see conventions below).
2. **Reference them in design sessions** — Claude sessions can read images from this folder for context.
3. **Update when direction changes** — keep old files with a `-v1`, `-v2` suffix if you need version history.

---

## Naming Conventions

| File name | What it shows |
|---|---|
| `homepage.png` | Full homepage — hero, cards, stats, featured members |
| `members-directory.png` | `/members` page — search/filter sidebar + member list |
| `member-profile.png` | Individual member profile detail page |
| `article-list.png` | `/articles` listing page |
| `article-detail.png` | Single article detail page |
| `events.png` | `/events` listing page |
| `auth.png` | `/auth` sign-in page |
| `application.png` | `/application` membership application flow |
| `member-dashboard.png` | `/member/dashboard` logged-in member home |
| `ops-dashboard.png` | `/ops` admin/ops dashboard |
| `navbar-desktop.png` | Navbar — desktop, logged-out state |
| `navbar-mobile.png` | Navbar — mobile menu expanded |
